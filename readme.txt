Project structure

app.js -  Main entry
models - The model data (ORM Models)
public - The public directory that contains all static content
         (images, style sheets, client-side JavaScript)
routes - The route definitions and implementations
services - The standalone services (Database service, Email service)
tests - Unit tests (testing is done through Jest)
views - Contains view-templates (using any templating language supported by express)


To download:

1. Clone the repository
2. Navigate to the directory

To run:

1. Install the required dependencies with npm install
2. Run the app with the command npm start
3. View app by viewing http://localhost:3000/ in your browser
